# SeaWorld

##Правила жизни в морском мире
###Организмы
####Движения

- Все организмы могут перемещаться только на одну клетку вперед
- Доступные направления движения по горизонтали и вертикали
- По диагонали ходить запрещено

###Пингвины
####Движения

- Каждый последующий шаг пингвины пытаются плавать
- Для движения пингвины выбирают случайное направление (вверх, вниз, вправо, влево)
- Если в указанном направлении пустое место, то пингвин должен переместиться туда

####Размножение
- Если пингвин остается жить в мире 3 хода подряд, то он должен попытаться вывести потомство
- Пингвины размножаются с помощью создания нового пингвина в свободном близ прилегающем месте от пингвина
- Если место для размножения нет, то пингвин не может продолжить свой род в этом ходу
- Для следующей попытки принести потомство пингвин должен ждать еще 3 хода

###Касатки
####Движения
- Каждый последующий шаг касатки плавают
- Касатки проверяют все направлениях и если встречается пингвин в одном из направлении, то касатка плывет туда и съедает пингвина
- Если в близ лежащем направлении нет пингвинов, то касатка плавает также как пингвин
- Касатки не могут есть друг друга

####Размножение
- Через каждые 8 ходов касатки пытаются вывести потомство
- Размножение происходит таким же образом как и у пингвинов

####Гибель
- Если касатка не поймает нового пингвина в течении 3 ходов, то касатка умирает, исчезает из водного мира
